# JUST Frontend

This is the frontend for the JUST application deployed on Vercel.

## Features
- User authentication (login/register)
- Incident reporting
- Interactive map
- Admin dashboard

## Deployment
- Hosted on Vercel
- Uses Vercel Functions for API
- Connected to Supabase database
